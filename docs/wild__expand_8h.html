Expands the W_ALL wildcard object into all of the objects (0 -> nr_swarm_robots) except my_symbolic_id in all of the structures of the Pcolony E.g for obj_with_any[OBJECT_ID_B_W_ALL] and nr_swarm_robots = 3 then OBJECT_ID_B_W_ALL -> [OBJECT_ID_B_0, OBJECT_ID_B_1, OBJECT_ID_B_2] in Pcolony.env, Pswarm.global_env, any Agent.obj and in any rule that contains the wildcarded object.

Parameters
pcolThe Pcolony where the expansion takes place
obj_with_any[]The array of objects that contain the W_ALL wildcard
is_obj_with_any_followed_by_id[]An array that specifies (with 0 / 1) whether each element in obj_with_any[] is or not followed by and W_ID object
obj_with_any_sizeThe size (number of elements) in the obj_with_any array (applies to is_obj_with_any_followed_by_id[])
my_symbolic_idThe computed symbolic id of this robot
nr_swarm_robotsThe total number of swarm robots. This number is used for the actual expansion (0 .. nr_swarm_robots - 1)
